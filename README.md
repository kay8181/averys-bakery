Bakery Admin System
A backend‑focused administrative platform for managing bakery operations with an emphasis on payments and deposits. This README outlines the system’s purpose, core financial features, setup, and workflow expectations for user.

Overview
The Bakery Admin System is designed for bakery owners, managers, and finance staff who need a centralized dashboard to manage orders, track payments, process deposits, and maintain accurate financial records. The system supports both in‑store and online transactions, ensuring that all monetary activity is captured, and auditable.

Core Functionalities (Admin‑Focused)
Payment Management
Payment Recording — Log payments received through online checkout, POS terminals, or manual entry.

Deposit 
Daily Deposit Logging — Record end‑of‑day cash deposits made to the bank.

Financial Reporting
Sales Summary Reports — month to date and year to date revenue breakdowns.

Deposit History — View all deposits with timestamps, amounts and vendor.

Generate Reports — Generate CSV summaries for accounting systems.

Order Oversight
Order List Management — View all orders with payment status.
Customer Payment History — View each customer’s past transactions and outstanding balances.

User Permissions
Payment and deposit management
